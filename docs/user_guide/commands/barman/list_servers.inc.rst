.. _commands-barman-list-servers:

``barman list-servers``
"""""""""""""""""""""""

Synopsis
^^^^^^^^

.. code-block:: text
    
    list-servers
        [ { -h | --help } ]
        [ --minimal ]

Description
^^^^^^^^^^^

Display all configured servers along with their descriptions.

Parameters
^^^^^^^^^^

``-h`` / ``--help``
    Show a help message and exit. Provides information about command usage.

``--minimal``
    Machine readable output.
