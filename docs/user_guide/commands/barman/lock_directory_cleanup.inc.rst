``barman lock-directory-cleanup``
"""""""""""""""""""""""""""""""""

Synopsis
^^^^^^^^

.. code-block:: text
    
    lock-directory-cleanup
        [ { -h | --help } ]

Description
^^^^^^^^^^^

Automatically removes unused lock files from the ``barman_lock_directory``.

Parameters
^^^^^^^^^^

``-h`` / ``--help``
    Show a help message and exit. Provides information about command usage.
